# Sales Executive Dashboard

Управленческая аналитика для руководителя отдела продаж на `Streamlit`.

Приложение строит KPI по план/факту, показывает состояние воронки, эффективность менеджеров, клиентскую концентрацию и прогноз выполнения периода.

## Что умеет приложение

- Управленческая сводка KPI: план, факт, выполнение плана, конверсия, win rate, средний чек.
- Открытый и взвешенный пайплайн с прогнозом достижения плана.
- Сравнение выбранного периода с предыдущим периодом такой же длины.
- Воронка продаж по стадиям: количество, потенциал, конверсия между стадиями.
- Аналитика менеджеров: план/факт, win rate, карта эффективности.
- Аналитика клиентов и продуктов: топ по выручке и конверсии.
- Реестр сделок с поиском, сортировкой и выгрузкой в CSV.
- Загрузка своего `.xlsx` через интерфейс или использование `sales_data.xlsx` по умолчанию.

## Требования к данным

Excel-файл должен содержать столбцы:

- `Дата сделки`
- `Менеджер`
- `Стадия сделки`
- `ФИО Клиента`
- `Продукт`
- `Сумма продажи план`
- `Сумма продажи факт`

## Запуск локально

1. Создай и активируй виртуальное окружение:

```bash
python3 -m venv .venv
source .venv/bin/activate  # macOS/Linux
```

2. Установи зависимости:

```bash
pip install -r requirements.txt
```

3. Запусти приложение:

```bash
streamlit run app.py
```

4. Открой URL из терминала (обычно `http://localhost:8501`).
